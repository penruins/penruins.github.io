通知

博思软件

2020年4月18日

当前的就业形势？

  1. 新冠疫情
  2. 经济下行

学习目标

  1. 自我职业定位
  2. 如何撰写一份好的简历
  3. 如何通过笔试通过率
  4. 如何提升面试通过率

企业信息

  1. 企业信息：资本情況、行业背景、市场地位、文化理念
  2. 信用查询：网络搜索器、企查查、启信宝、天眼查等企业信用信息查询工具信息披露：巨潮资讯网、中国证监会网站
  3. 其他信息：企业官网、企业内部在职人员（学长学姐）

岗位选择

  1. 专业
  2. 兴趣
  3. 个性地点
  4. 行业
  5. 新酬

简历撰写误区

  1. 重设计轻内容
  2. 夸大性描述
  3. 内容占比不合适
  4. 一份简历投递多个岗位

一份完整的简历包括哪些基本结构？

  1. 简历封面/标题
  2. 基本信息（姓名、出生年月、学校、专业、户口所在地、现居住地、手机、电子邮箱、个人相片可选：政治面貌等）
  3. 个人经历（社团经历、兼职或实习经历、学习经历、生活经历等）
  4. 其他信息（个人荣誉类、兴趣爱好、自我评价）加分项
    注意：排版简洁，重点突出，一页纸

笔试考点分析
笔试的基本考察点：专业知识和思维能力

其他准备工作

  1. 专业课程：针对岗位要求，强化相关知识
  2. 强化训练：通过牛客网、应届生求职论坛等平台，进行强化训练
  3. 遵守考场纪律
  4. 先做擅长的，保证答题率及准确率
  5. 题目尽量不留白
  6. 笔试后，保持手机通畅，确保能收到面试信息

自我介绍（三分钟内）

  1. 我是谁？
  2. 应聘的职位是什么？
  3. 对公司的基本情况及业务的认知
  4. 我有哪些经历与岗位比较一致？我的优势是什么？
  5. 与岗位无关的兴趣爱好内容尽量不说
  6. 用简洁的方式突出自己的能力优势

面试礼仪
举止：

  1. 见面时。主动打招呼。彬彬有礼，大方得体
  2. 面试时。目光聚焦，展现出自信及面试官的尊重交谈时，可配合一些手势讲解，但不要频繁耸肩、手舞足蹈
    着装
  3. 通常情况，互联网行业只要着装整洁就好。
  4. 面试前，务必提前做好检查，根据企业的特性和文化准备着装

面试评估流程&方式
技术面
考察点：由部门的技术经理或专家担任面试官。会依据简历上的项目经历、技能点进行深入的沟通，以此评估候选人的专业胜任力。通过情景式提问，评估解決题能力及创新能力。
注意点：依据STAR法则介绍项目经历，在准备技术向题及项目问题时，切忌泛泛而谈。重点闻述项目的技术细节点。遇到不会的技术问题，不要紧张，正常作答。

HR面
考察点：职业规划、求职动机目标感和自我驱动力、企业文化和团队匹配
注意点：清晰的自我认知。合理的薪酬期望。正确的求职态度，每一次面试都是加深自我认识，不断成长的过程。
