  • 使用@Validated校验List接口参数的两种方式

@AssertTrue 方法名前缀 ​

  • 使用 @AssertTrue 进行字段校验时，方法名称必须以 is 或 has 开头, 不然不会生效
    • Tips: 在使用 swagger2 的项目中，如果使用 is 开头，会在接口文档中将方法解析为一个字段，换成 has 为前缀之后不会出现这样的情况
      • 如果一定要使用 is 作为方法名称的前缀，应该也可以使用 @ApiModelProperty(hidden = true) 将其隐藏
java

@AssertTrue(message = "计划培训开始时间不能大于计划培训结束时间")
public boolean isTrainEndTimeAfterTrainStarTime() {
    return trainStartTime.compareTo(trainEndTime) <= 0;
}

校验整数字段不能为空 ​

java
@NotNull(message = "pageNum 不能为空")
private Integer pageNum;

校验字符串不能为空 ​

java
@NotBlank(message = "申请人 不能为空")
private String applicant;

校验时间段字段先后关系 ​

java
// 日期为 字符串 类型
@NotEmpty(message = "计划培训开始时间 不能为空")
private String trainStartTime;

@NotEmpty(message = "计划培训结束时间 不能为空")
private String trainEndTime;

@AssertTrue(message = "计划培训开始时间不能大于计划培训结束时间")
public boolean isTrainEndTimeAfterTrainStarTime() { // 方法名称要以 is 或 has 开头
    return trainStartTime.compareTo(trainEndTime) <= 0;
}
java
// 日期为 LocalDate 类型
@NotNull(message = "开始日期(startDate) 不能为空")
private LocalDate startDate;

@NotNull(message = "结束日期(endDate) 不能为空")
private LocalDate endDate;

@AssertTrue(message = "结束日期(endDate) 不能先于 开始日期(startDate)")
public boolean isEndDateAfterStartDate() {
    return !endDate.isBefore(startDate);
}
