在 Hugo 中配置 TOC 从一级标题开始显示 ​

要在 Hugo 中让侧边栏的目录(TOC)从一级标题(h1)开始显示，你可以通过以下几种方法实现：

方法一：修改 config.toml ​

在你的 Hugo 配置文件(通常是 config.toml 或 config.yaml)中添加或修改以下设置：

toml
[markup]
[markup.tableOfContents]
startLevel = 1
endLevel = 6

或者对于 YAML 格式：

yaml
markup:
  tableOfContents:
    startLevel: 1
    endLevel: 6

方法二：模板中指定 ​

如果你在模板中调用 TOC，可以这样指定：

html
{{ .TableOfContents }}

默认情况下，Hugo 会从二级标题(h2)开始显示 TOC。要修改这个行为，你需要使用方法一中的配置。

方法三：使用自定义 CSS ​

如果配置修改后仍然不符合预期，你可以添加自定义 CSS 来隐藏不需要的层级：

css
/* 隐藏除一级标题外的其他层级 */
#TableOfContents ul ul {
    display: none;
}

方法四：使用 Goldmark 配置 ​

如果你使用的是 Goldmark 渲染器(默认)，确保配置正确：

toml
[markup.goldmark]
[markup.goldmark.parser]
[markup.goldmark.parser.attribute]
block = true
title = true

注意事项 ​

  1. 确保你的内容中确实有一级标题(# 或 <h1>)
  2. 某些主题可能会覆盖默认的 TOC 行为，你可能需要检查主题文档
  3. 清除缓存(hugo --gc)有时能解决显示问题

如果你使用的是特定主题，可能需要查看该主题的文档，因为有些主题会提供自己的 TOC 配置选项。
