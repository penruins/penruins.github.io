  3. 文件目录权限的含义是啥 dr-xr-xr-x.
    • d : 第一位表示文件类型。d是目录文件，l是链接文件，-是普通文件，p是管道
    • r-x : 第5-7位表示和这个文件属主所在同一个组的用户所具有的权限
    • r-x : 第8-10位表示其他用户所具有的权限

  19. docker desktop 下面展示的 `RAM` 是什么?
    • `Random Access Memory` `随机存取存储器` 也叫主存，是与 CPU 直接交换数据的内部存储器。它可以随时读写（刷新时除外），而且速度很快，通常作为操作系统或其他正在运行中的程序的临时数据存储介质。RAM工作时可以随时从任何一个指定的地址写入（存入）或读出（取出）信息。它与ROM的最大区别是数据的易失性，即一旦断电所存储的数据将随之丢失。RAM在计算机和数字系统中用来暂时存储程序、数据和中间结果。
