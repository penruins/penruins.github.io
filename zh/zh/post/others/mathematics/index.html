    什么是偏导数 2025-02-08 #

    偏导数是多元函数对其中一个自变量的导数，用于衡量函数在该自变量方向上的变化率。具体来说，对于函数 ( f(x_1, x_2, \dots, x_n) )，其关于 ( x_i ) 的偏导数记作 ( \frac{\partial f}{\partial x_i} )，表示在其他变量保持不变时，函数 ( f ) 随 ( x_i ) 的变化率。

    定义 #

    设 ( f(x_1, x_2, \dots, x_n) ) 是一个多元函数，( x_i ) 是其中一个自变量。( f ) 关于 ( x_i ) 的偏导数定义为：

    [ \frac{\partial f}{\partial x_i} = \lim_{h \to 0} \frac{f(x_1, \dots, x_i + h, \dots, x_n) - f(x_1, \dots, x_i, \dots, x_n)}{h} ]

    几何意义 #

    偏导数 ( \frac{\partial f}{\partial x_i} ) 表示函数 ( f ) 在 ( x_i ) 方向上的切线斜率。

    计算方法 #

    计算偏导数时，将其他变量视为常数，仅对目标变量求导。例如，对于 ( f(x, y) = x^2 y + y^3 )，关于 ( x ) 的偏导数为：

    [ \frac{\partial f}{\partial x} = 2xy ]

    关于 ( y ) 的偏导数为：

    \[ \[ \frac{\partial f}{\partial y} = x^2 + 3y^2 \] \]

    应用 #

    偏导数在优化、物理、经济学等领域有广泛应用，如梯度计算、求解极值点等。

    总结 #

    偏导数是多元函数对某一自变量的导数，反映函数在该方向上的变化率，计算时将其他变量视为常数。

    \[ f(x) = \int_{-\infty}^\infty\hat f(\xi)\,e^{2 \pi i \xi x}\,d\xi \]
