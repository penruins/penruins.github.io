    Spring Cloud VS Spring Cloud Alibaba #

    • 现在学spring cloud alibaba 还是spring cloud?

    以下是 Spring Cloud 和 Spring Cloud Alibaba 的详细对比，从技术定位、核心组件、适用场景等维度综合分析，并结合实际应用场景提供选型建议：

    功能 Spring Cloud Spring Cloud Alibaba 优势对比
    服务注册与发现 Eureka（AP，已停更） Nacos（AP/CP 双模，高性能） Nacos 支持动态配置管理，性能提升 50%+
    配置中心 Config Server（基于 Git） Nacos Config（可视化界面） Nacos 支持实时推送和多环境配置
    流量控制与熔断 Hystrix（维护模式） Sentinel（动态规则、实时监控） Sentinel 支持精细化流量控制和系统保护
    分布式事务 无官方方案（依赖第三方） Seata（原生集成 AT/TCC 模式） Seata 简化分布式事务开发
    消息队列 无（依赖 RabbitMQ） RocketMQ（高吞吐、事务消息） RocketMQ 适合高并发和削峰填谷场景
