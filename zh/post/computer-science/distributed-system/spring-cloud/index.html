    Spring Cloud #

    服务注册与发现 #

    Eureka #

    • Netflix 开源的服务注册中心，采用 AP 架构（高可用性），通过心跳机制检测服务状态
    • 包含 Eureka Server（注册中心）和 Eureka Client（服务提供者/消费者）
    • 注：Eureka 2.x 已停止维护，建议新项目使用 Nacos

    Nacos #

    • Spring Cloud Alibaba 的核心组件，兼具服务注册中心和配置中心功能，支持 CP/AP 模式切换
    • 提供动态服务发现、健康检查、配置管理等功能，适用于云原生场景

    服务调用与负载均衡 #

    Feign #

    • 声明式的 REST 客户端，通过动态代理简化服务间 HTTP 调用，整合了 Ribbon 实现负载均衡

    Ribbon #

    • 客户端负载均衡组件，支持轮询、随机、权重等算法，需配合 Eureka/Nacos 使用
    • 注：Ribbon 已进入维护阶段，Spring Cloud LoadBalancer 是新一代替代方案

    Dubbo #

    • Spring Cloud Alibaba 支持的 RPC 框架，适用于高性能服务调用，替代 Feign+Ribbon 的组合

    服务容错与熔断 #

    Hystrix #

    • Netflix 开源的熔断器，通过熔断、降级、隔离机制防止服务雪崩
    • 注：Hystrix 已停止维护，推荐使用 Sentinel

    Sentinel #

    • Alibaba 开源的流量控制组件，支持熔断、限流、系统自适应保护，提供实时监控

    服务网关 #

    Zuul #

    • Netflix 的 API 网关，负责路由、过滤、负载均衡，但性能较低
    • 注：Zuul 1.x 已停止维护，Zuul 2.x 未集成到 Spring Cloud

    Spring Cloud Gateway #

    • 基于 Reactor 的高性能网关，支持动态路由、限流、集成 Sentinel

    配置中心 #

    Spring Cloud Config #

    • 集中化管理配置，支持 Git、本地存储等，但需配合 Bus 实现动态刷新

    Nacos Config #

    • 与 Nacos 注册中心无缝集成，支持配置动态更新和版本管理

    消息总线与事件驱动 #

    Spring Cloud Bus #

    • 通过消息队列（如 RabbitMQ、Kafka）广播配置变更，实现配置热更新

    Spring Cloud Stream #

    • 统一消息中间件抽象层，支持 Kafka、RocketMQ 等

    分布式链路追踪 #

    Sleuth + Zipkin #

    • 追踪微服务调用链路，生成请求依赖图，便于故障排查

    SkyWalking #

    • 更强大的 APM 工具，支持分布式追踪、性能监控

    分布式事务 #

    Seata #

    • Alibaba 开源的分布式事务解决方案，支持 AT、TCC、SAGA 模式

    其他组件 #

    Spring Cloud Security #

    • OAuth2、JWT 等安全认证

    Spring Cloud OpenFeign #

    • Feign 的增强版，支持 Spring MVC 注解
