律师的作用不是维护正义公正，而是保护当事人利益，只要手段不是非法的即可。律师的必须无视良知和正义，它的正面作用是通过充分博弈实现的。博弈吗，跟良知无关。最好的律师是狼狗、狐狸、恶狼、毒蛇的混合体。

我想告诉你一切都会好起来，但并不会。要是有办法能让你好受点，我想就是习惯这种伤痛。我在卡斯珀参加过一个疗伤研讨会。你听说过吗？我不知道为什么，只是想让悲痛远离，只想为这个无解难题找个答案。讨论会结束后，辅导员来找我，坐我旁边，说了些很触动我的话。我不知道触动我的是他说的话，还是他说话的方式。他说"我有好消息，也有坏消息。坏消息是你回不去从前了，你不再完整，再无可能。你失去了女儿，她无可取代。好消息是，你一旦接受现实，不再抗拒这种痛楚，便能在心里与她重见，会记得所有她给的爱，她所知的一切快乐。"马丁，关键在于你不能回避这种痛楚。回避痛楚，你就剥夺了自己，剥夺了自己关于她的一切回忆，一无所剩，从她迈出的第一步，到她最后的微笑，全都抹去了。接受那份痛苦，马丁。听到了吗？感受它。只有这样，才能留她在你身边。

上善若水，水善利万物而不争。处众人之所恶，故几于道。居善地，心善渊，与善仁，言善信，正善治，事善能，动善时。夫唯不争，故无尤
The best way to do good is like water, which moistens everything but never fights with it. He is willing to be in the lowly place that people hate, so it is close to the Tao. To live, we should be as humble as water. We should be as deep-seated as water. We should treat people as water. We should be as intimate as water. We should talk as water. We should be as honest as water. We should be good at governance. We should be as omnipotent as water. We should act as water and seize the opportunity. It is precisely because he does not contend with things that he has no fault.

在世上，最让人畏惧的恰恰是通向自己的道路。 我常常幻想未来的景象，梦想自己可能会成为的角色，或许是诗人、预言者、画家等等。然而这些都不算什么。我存在的意义并不是为了写诗、预言或作画，任何人生存的意义都不应是这些。这些只是旁枝末节。对每个人而言，真正的职责只有一个：找到自我。无论他的归宿是诗人还是疯子，是先知还是罪犯——这些其实与他无关，毫不重要。他的职责只是找到自己的命运——而不是他人的命运——然后在心中坚守其一生，全心全意，永不停息。所有其他的路都是不完整的，是人的逃避方式，是对大众理想的懦弱回归，是随波逐流，是对内心的恐惧。
